EUROMOON - examenproject Programming Advanced 1

Euromoon is een applicatie in Javascript voor een Europese treinmaatschappij. 
Het systeem maakt het mogelijk dat passagiers zichzelf registreren, tickets kopen en hun reizen beheren. 
Het personeel is verantwoordelijk voor het aanmaken van reizen, het koppelen van treinen en het beheren van tickets.
Het project focust op het beheer van passagiers, personeel, treinen, reizen en tickets.

#Features:
- Passagiers registreren
- Personeel beheren (conducteur, stewards, bagagepersoneel)
- Treinen aanmaken en koppelen aan reizen
- Reizen plannen tussen Europese hoofdsteden
- Tickets verkopen en raadplegen
- Boardinglijst afdrukken naar tekstbestand


Opmerkingen:

Boardinglijst bevat alleen stations en datum, geen namen of e-mails

Project gebruikt Java List voor opslag van passagiers, tickets en reizen

Methodes die bestanden schrijven gebruiken throws IOException 

Tijdens de ontwikkeling is de automatische codecontrole van IntelliJ IDEA gebruikt om fouten en waarschuwingen te controleren

CHATGPT gebruikt voor methode aan te maken om boardingslijst af te drukken (PrintWriter)
